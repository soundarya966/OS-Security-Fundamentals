# Operating System Security Checklist

## User & Access Control
- [x] Use strong passwords
- [x] Avoid logging in as root
- [x] Use sudo only when required
- [x] Follow least privilege principle

## File Permissions
- [x] Check permissions using ls -l
- [x] Restrict write permissions for sensitive files
- [x] Change ownership using chown
- [x] Apply correct permissions using chmod

## Firewall & Network Security
- [x] Enable UFW firewall
- [x] Allow only required ports
- [x] Block unused inbound connections

## Process & Service Management
- [x] Identify running processes using ps and top
- [x] List services using systemctl
- [x] Disable unnecessary services

## System Hardening
- [x] Keep system updated
- [x] Remove unused applications
- [x] Reduce attack surface
- [x] Monitor system activity

## Conclusion
Applying these security measures helps protect the operating system from unauthorized access, malware, and attacks.
